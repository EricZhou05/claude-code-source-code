# 学习指南 — 如何系统学习本仓库并迁移到 Gemini 工作流

> 本学习指南专为中文使用者设计，帮助你从零开始系统学习 Claude Code v2.1.88 源码分析文档，并将所学知识迁移到你自己的 Gemini AI 工作流中。

---

## 本指南的目标

1. **系统性学习**：按正确顺序阅读和理解每一份泄露源码分析文档
2. **实践应用**：将文档中揭示的 AI 提示词设计原则用于自己的工作
3. **工作流迁移**：将 Claude Code 的优秀设计理念迁移到 Gemini（或任何 AI 工具）的工作流中

## 文档来源说明

所有分析文档均基于从 npm 包 `@anthropic-ai/claude-code` **2.1.88** 版本中反编译得到的 TypeScript 源码。这些分析文档揭示了：

- Anthropic 如何设计 AI 系统提示词（System Prompt）
- AI 产品的架构设计原则
- 提示词工程的实战技巧
- AI 行为控制的机制设计

这些知识对任何希望深入理解和运用 AI 工具（包括 Gemini）的人都极具价值。
